Chat UI Kits 聊天机器人情感化回应系统
简介
Chat UI Kits是基于认知耦合框架搭建的聊天机器人情感化回应系统设计，包含基础组件的应用以及情感化回应方式的创新组件设计。基于当前认知语言技术发展的背景，聊天机器人情感化回应系统用以解决技术限制之下聊天机器人答非所问、刻板回答所带来的体验问题，从认知层面出发探索更多情感体验的可能性，关注用户在会话式交互界面中的情感提升。在认知技术的基础上，通过视觉表达的方式，试图在消费场景中打造出贴心、个性化的消费型聊天机器人助手形象。
内容概览
Chat UI Kits包含设计规范、基础组件、对话组件、展示组件四部分主要内容。其中，设计规范定义对话容器、颜色、图标等基础使用规范；基础组件定义主交互区域的基础控件展示形式与使用规范；对话组件与展示组件在主交互区域基础的使用控件之外，拓展了消费型聊天机器人的情感回应方式，通过静态、动效相结合的展示方式，拓宽情感回应层次，丰富了聊天机器人的整体形象展示。
